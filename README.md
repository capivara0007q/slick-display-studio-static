# 🎯 Slick Display Studio - Versão Estática

Versão frontend-only do projeto **Slick Display Studio** preparada para deploy gratuito em plataformas como Netlify, GitHub Pages ou Vercel.

## 🚀 Deploy Rápido

### Via Netlify (Mais Rápido - 2 min)
1. Acesse [netlify.com](https://netlify.com)
2. Arraste a pasta `dist/client/` 
3. Pronto! 🎉

### Via GitHub Pages (Mais Permanente)
1. Crie repositório no GitHub
2. Suba o conteúdo de `dist/client/`
3. Ative Pages nas configurações

[➤ Ver instruções completas](./DEPLOY_INSTRUCOES.md)

## 📦 O que Está Incluído

- ✅ React + TanStack Router
- ✅ Radix UI + Tailwind CSS
- ✅ Todas as páginas e componentes
- ✅ Tamanho otimizado (~2.5 MB)
- ✅ Pronto para produção

## ⚠️ Limitações

- ❌ Login (sem backend)
- ❌ Dados dinâmicos (sem Supabase)
- ❌ APIs (sem servidor)

**Ideal para**: Demo, portfolio, visualização de UI

